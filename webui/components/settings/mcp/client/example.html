Synapse использует стандартную конфигурацию JSON, знакомую по другим ИИ-приложениям.
Конфигурация представляет собой JSON-объект, содержащий объект "mcpServers", где каждый ключ — это отдельный MCP-сервер.
Локальные серверы определяются переменными "command", "args", "env".
Удаленные серверы определяются переменными "url", "headers".
Для отключения сервера без удаления конфигурации можно установить "disabled" в значение true.
Можно задать пользовательское описание "description", чтобы предоставить A0 дополнительную информацию о сервере.
Все серверы также могут определять "init_timeout" и "tool_timeout", которые переопределяют глобальные настройки.
